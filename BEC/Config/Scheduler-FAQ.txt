Вопрос: Я не понимаю как использовать формат времени.
		Что будет при ччммсс и чч:мм:сс В чем разница? 
	
Ответ:  BEC использует каждый формат времени по разному.
   Формат времени ЧЧ:ММ:СС используется для выполнения чего-либо в указанное время например 18:00:00
   Формат времени ЧЧММСС используется как таймер обратного отсчета с повтором. Например 001500 - Каждые 15 минут будет что-то просиходить.

Вопрос: Почему не работает Планировщик?

Ответ: 1. Убедитесь что Планировщик включен в Config.cfg
	   2. Если вы редактировали планировщик, то скорее всего допустили где-то ошибку.
	   В папке Logs/Error создается файл, где сказано в чем ошибка.

Вопрос: Я хочу чтобы первое сообщение появилось через 20 секунд, но второе через 30 секунд. Данный процесс должен повторяться каждые 20 секунд.

Ответ:

	<job id='0'>
		<day>1,2,3,4,5,6,7</day>
		<start>000020</start>			// Время через которое появиться первое сообщение
		<runtime>000020</runtime>		// Время повтора сообщения
		<loop>-1</loop>					// Сколько раз повторять -1 - постоянно
		<cmd>say -1 text 1</cmd>		// Какой текст будет высвечиваться
	</job>
	<job id='1'>
		<day>1,2,3,4,5,6,7</day>
		<start>000030</start>
		<runtime>000020</runtime>
		<loop>-1</loop>
		<cmd>say -1 text 2</cmd>
	</job>	

  18:00:20 : RCon admin #0: (Global) text 1
  18:00:30 : RCon admin #0: (Global) text 2
  18:00:40 : RCon admin #0: (Global) text 1
  18:00:50 : RCon admin #0: (Global) text 2

Вопрос: Какие команды поддерживает Планировщик?

Ответ: Все команды вписываются в <cmd> тег.
Список команд

say -1 Текст
loadbans
loadscripts
loadevents
writebans
maxping 250
#shutdown
#restart
#reassign
#lock
#unlock
#mission filename (Загрузка .pbo файла)
systime
uptime
fortune
